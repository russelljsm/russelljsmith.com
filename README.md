# Russell J. Smith Personal Website

Static personal website for `russelljsmith.com`.

## Publishing

This site is intended for GitHub Pages.

- Production domain: `russelljsmith.com`
- GitHub Pages custom-domain file: `CNAME`
- Static files: `index.html`, `styles.css`, `script.js`, `assets/`

Future updates can be made by editing the static files and pushing the repository to GitHub.
